# HexForge 未完成功能开发清单

## 一、编辑功能

### 1.1 撤销/重做 (Undo/Redo) ✅ 已完成
**需求描述：**
- 支持 50+ 级撤销历史
- 支持 Ctrl+Z 撤销、Ctrl+Y 重做
- 记录以下操作：字节修改、插入、删除、批量填充
- 撤销栈需要持久化（可选）

**技术实现：**
- 使用 Command Pattern
- 在 UndoStack 中存储可序列化的命令对象
- 支持跨会话撤销（可选）

---

### 1.2 批量修改 (Batch Edit) ✅ 已完成
**需求描述：**
- **填充功能**：选中区域填充为指定字节值
  - 填充为 00 或 FF
  - 填充为自定义值（如 AB CD）
- **递增/递减**：
  - 选中区域的字节值逐个 +1/-1
  - 支持有符号/无符号整数溢出处理
- **取反**：选中区域按位取反
- **反转**：选中区域的字节顺序反转

**技术实现：**
- 添加 EditMenu 菜单项
- 创建 BatchEditDialog 对话框
- 支持预览模式

---

### 1.3 跳转历史 (Jump History) ✅ 已完成
**需求描述：**
- 记录所有跳转操作（Goto、书签、双击等）
- 支持返回上一步/下一步
- 显示历史列表面板
- Ctrl+Alt+Left 返回上一步
- Ctrl+Alt+Right 前进一步

**技术实现：**
- 使用双向链表或栈记录历史

- 自动检测文件类型并应用模板
- 模板字段高亮显示

**技术实现：**

- 添加自动检测逻辑
